# A/B-тест: влияние push-уведомлений на активность пользователей

## Контекст
Продукт: мобильное фитнес-приложение  

Задача: повысить вовлечённость пользователей.  
В качестве метрики используется средний пробег (км), как прокси пользовательской активности и потенциального удержания.

## Цель
Оценить влияние новой механики push-уведомлений на средний пробег пользователей.

## Гипотеза
Отправка мотивационного push-уведомления с ссылкой после достижения определённой дистанции увеличит средний пробег пользователя, так как стимулирует продолжение активности.

## Дизайн эксперимента
- Тип: A/B тест  
- Группы: Control / Test  
- Метрика: средний пробег (км)  
- Единица анализа: пользователь  

## Проверка данных
- Проверено отсутствие пересечения пользователей между группами  
- Проведена базовая очистка данных  

## Результаты
- T-test: p-value = 0.78  
- Bootstrap доверительный интервал разности средних: [-1.42; 1.98]  

Статистически значимых различий между группами не обнаружено.

## Интерпретация
Новая механика push-уведомлений не оказывает значимого влияния на средний пробег пользователей.

Полученный эффект либо отсутствует, либо слишком мал для обнаружения на текущем объёме данных.

## Анализ чувствительности
- MDE (минимально детектируемый эффект): 2.59 км  

Текущий объём выборки позволяет выявлять только крупные изменения.  
Более слабые эффекты (например, 0.5–1 км) могли остаться незамеченными.

## Решение
Не внедрять новую механику push-уведомлений в текущем виде.

## Рекомендации
- Переработать механику уведомлений (контент, тайминг, триггеры)  
- Увеличить длительность теста для снижения MDE  
- Провести анализ по сегментам пользователей  

## Технологии
- Python  
- pandas, numpy  
- scipy  
- statsmodels

## Пример кода (Bootstrap)
```python
def confidense_bootstrap(sample1, sample2, boot_iters = 1000):
    final_sample = []
    for i in range(boot_iters):
        mean1 = np.random.choice(sample1, len(sample1)).mean()
        mean2 = np.random.choice(sample2, len(sample2)).mean()
        final_sample.append(mean1 - mean2)
    return np.percentile(final_sample, 2.5), np.percentile(final_sample, 97.5)
```
